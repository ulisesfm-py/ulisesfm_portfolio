Projects

Unet Model for Land Image Segmentation

This project involved developing and comparing three distinct models — a U-Net neural network, a Logistic Regression model, and a Random Forest Classifier — for land segmentation classification using aerial imagery data from Kaggle. The project aimed to classify land use types such as water, land, roads, buildings, and vegetation by leveraging both deep learning and traditional machine learning methods. The U-Net model was trained with a custom loss function (Categorical Focal Loss), optimized with GPU support for efficient computation. Meanwhile, Logistic Regression and Random Forest Classifiers were trained using pixel-wise data to benchmark performance against the U-Net model. The project demonstrated effective segmentation with an emphasis on model accuracy, class balance, and computational efficiency.

Military Aircraft Classification with CNNs

In this project, I developed and compared two convolutional neural network models—EfficientNetB3 and ResNet50—for classifying images of military aircraft. The dataset included various types of aircraft, and the goal was to accurately identify the model of an aircraft from its image. The project involved data preprocessing, model training with custom callbacks, and performance evaluation using metrics like precision, recall, and F1-score.

Bulldozer Exploratory Data Analysis

This project involves conducting an exploratory data analysis and building a predictive model for the sale price of bulldozers using historical sales data. Leveraging machine learning techniques such as Random Forest Regression and libraries like Scikit-Learn, Pandas, and Seaborn, the project focuses on identifying key features and trends that influence bulldozer prices. The dataset, sourced from Kaggle, includes time-series data with attributes like model type, sale date, and equipment characteristics. The goal is to provide insights and develop a robust model for accurate price predictions.
